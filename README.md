# タブマネージャー for Zenn
タブマネージャー for Zennは開きすぎたZennのタブを一つにまとめてくれるChromeの拡張機能です。<br />
タブをグループにしたり、ポップアップでタイトルを一覧で表示することができます！<br />
また、Zennのタブを開いていない時、ポップアップから簡単にZennにアクセスできます。
